
This file contains two sheets. Sheet1 contains validity of bands with respect to Modes and BandNumber.

0 is indicates invalid/less confidence. 1 indicates valid band.


Reason for assigning invalid/less confidence tag to a band is given in Sheet 2.


INVALID: Extremely low or no response from the instrument. This region of spectrum is updated with the help of neighboring bands.

OSF : Order Sorting Filter Location: Less confidence in radiance

SATURATION : Invalid band as spectrometer has saturated

VALID :  Good Band (where user can expect image content)
